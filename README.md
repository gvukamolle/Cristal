# Claude Rock

**Claude прямо в Obsidian.** Полноценный ИИ-ассистент для работы с заметками — без ограничений веб-интерфейса.

---

## Что это?

Claude Rock — это плагин для Obsidian, который интегрирует Claude (Anthropic) прямо в ваше хранилище заметок. Не просто чат-бот, а полноценный ассистент, который **видит ваши заметки**, может их **редактировать**, **искать информацию в интернете** и работать с контекстом вашей базы знаний.

### Почему это круто?

- **Claude работает с вашими заметками напрямую** — упоминайте файлы через `@`, и Claude увидит их содержимое
- **Редактирование в один клик** — ответы Claude можно сразу вставить в заметку (заменить, добавить в конец, скопировать)
- **Веб-поиск и анализ ссылок** — Claude может гуглить и читать веб-страницы
- **8 языков интерфейса** — русский, английский, французский, немецкий, испанский, хинди, китайский, японский
- **Слеш-команды** — быстрые действия: `/summarize`, `/translate`, `/rewrite` и другие
- **Глубокое мышление** — режим extended thinking для сложных задач
- **Выбор модели** — Sonnet, Opus или Haiku под разные задачи

## Возможности

### Работа с заметками
- Упоминание файлов через `@` — Claude видит содержимое
- Прикрепление файлов и изображений
- Кнопки "Заменить", "Добавить", "Копировать" для быстрой вставки ответов
- Создание новых заметок из ответов Claude

### Встроенные команды
| Команда | Описание |
|---------|----------|
| `/summarize` | Краткое резюме заметки |
| `/translate` | Перевод на указанный язык |
| `/rewrite` | Переписать понятнее |
| `/expand` | Развить и дополнить идеи |
| `/fix` | Исправить ошибки и опечатки |
| `/bullets` | Преобразовать в список |
| `/questions` | Сгенерировать вопросы по теме |
| `/difficulty` | Объяснить проще или сложнее |

### Настройки
- **Путь к CLI** — если Claude установлен нестандартно
- **Язык ассистента** — выбор из 8 языков
- **Системные инструкции** — файл `CLAUDE.md` для кастомизации поведения
- **Модель по умолчанию** — Sonnet, Opus или Haiku
- **Режим глубокого мышления** — для сложных задач
- **Разрешения** — веб-поиск, загрузка страниц, субагенты
- **Кастомные команды** — создавайте свои слеш-команды

## Установка

1. Скачайте репозиторий или клонируйте его в `~/vault/.obsidian/plugins/`
2. Включите плагин в настройках Obsidian: **Настройки → Сторонние плагины → Claude Rock**
3. Установите [Claude CLI](https://docs.anthropic.com/en/docs/claude-code) и авторизуйтесь:
   ```bash
   npm i -g @anthropic-ai/claude-code
   claude
   ```

## Использование

### Базовое
Просто пишите в чат — Claude ответит. Используйте `@` для упоминания файлов из вашего хранилища.

### Команды
Начните сообщение с `/` чтобы увидеть доступные команды. Например:
- `/summarize` — сделать резюме текущей заметки
- `/translate английский` — перевести на английский

### Контекст
- **@файл.md** — Claude увидит содержимое файла
- **Прикрепление файлов** — перетащите файл в чат или используйте кнопку
- **Выделенный текст** — если выделить текст в заметке перед отправкой, нажмите правой кнопкой мыши и выберите "Claude Rock: Упомянуть при запросе", Claude увидит его и будет работать только с выделенный куском текста

### Кнопки ответа
После ответа Claude появляются кнопки:
- **Заменить** — заменить содержимое текущей заметки
- **Добавить** — добавить в конец заметки
- **Копировать** — скопировать в буфер обмена
- **Новая страница** — создать новую заметку с ответом

## Настройка

Откройте **Настройки → Claude Rock** для конфигурации:

- **Путь к CLI** — обычно просто `claude`, если установлен глобально
- **Язык ассистента** — влияет на язык ответов и интерфейса
- **Системные инструкции** — редактируйте `CLAUDE.md` для кастомизации
- **Модель** — выбирайте под задачу (Sonnet для баланса, Opus для сложного, Haiku для быстрого)
- **Глубокое мышление** — включите для задач, требующих анализа
- **Разрешения** — контролируйте, что может делать Claude

---

## Решение проблем

- **CLI не найден** — запустите `npm i -g @anthropic-ai/claude-code` и проверьте `claude --version`
- **Плагин не появляется** — перезагрузите Obsidian (`Cmd/Ctrl + R`)
- **Ошибка авторизации** — запустите `claude` в терминале и авторизуйтесь заново
